# Finnish Vocabulary Trainer

## Description

This project is a console application written in C#.  
The program helps users practice Finnish rections.

The application allows the user to add new words, remove words, add rections with example sentences, and practice vocabulary.  
All data is stored in text files.

This project was created as a programming course assignment.

---

## Features

- Add new Finnish words
- Delete words from the list
- Add rections with example sentences
- Delete rections
- View lists of saved words and rections
- Random vocabulary practice
- Input validation to prevent incorrect input

---

## Technologies

- C#
- .NET Console Application
- File handling (reading and writing text files)

---

## How to Run

1. Download or clone this repository.
2. Open the project in **Visual Studio**.
3. Build the solution.
4. Run the program.

The application runs in the console.

## Project Structure

Program.cs – main program logic  
Input.cs – user input handling
